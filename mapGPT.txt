[주요 클래스 관계 요약]
================================================================================

1. Controller → Service → Repository → Entity
   예: OrderController → OrderService → OrderRepository → Order

2. Service 간 의존성
   - OrderService → InventoryService, DeliverySchedulingService
   - OrderChangeRequestService → InventoryService, PaymentService
   - VoiceOrderCheckoutService → OrderService

3. Entity 간 관계
   - Order 1:N OrderItem
   - Order 1:N OrderChangeRequest
   - OrderChangeRequest 1:N OrderChangeRequestItem
   - Order 1:1 DeliverySchedule
   - User N:1 Order (userId)
   - MenuItem N:1 OrderItem (menuItemId)

4. 데이터베이스 분리
   - Main DB (mrdabak.db): User, DinnerType, MenuItem, DinnerMenuItem
   - Order DB (orders.db): Order, OrderItem, OrderChangeRequest, OrderChangeRequestItem
   - Schedule DB (schedule.db): DeliverySchedule, EmployeeWorkAssignment
   - Inventory DB (inventory.db): MenuInventory, InventoryReservation

5. 외부 의존성
   - Spring Boot (Web, Data JPA, Security, Validation)
   - JWT (io.jsonwebtoken)
   - SQLite (org.xerial.sqlite)
